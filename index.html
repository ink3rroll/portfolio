Portfolio

Replicated Google Advanced Search Page

This web page is a replica of Google's advanced search page. Its features use get requests to properly display what the user intends to search. It lets the user input important, must-have keywords, and can exclude specific keywords and ranges to the search query. I used html, css, python and django library to make this project.

Travel Log Data Entry App

This data entry app generates receipts for every travel made by the employees. The company pays the expenses and having receipts is a must. It makes the generation of the receipt automatic and fast. I made this program using python and tkinter library.

Boolean Algebra Simplification Calculator

This command-line app simplifies complex boolean expressions. It uses a parsing library called ply, a python lex and yacc library, for its parsing mechanic.
